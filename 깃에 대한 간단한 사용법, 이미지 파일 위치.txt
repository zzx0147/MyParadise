2.SourceTree 사용법

윈도우 검색창에서 SourceTree를 검색해서 실행한 뒤

사용하기 전에 항상 Pull 버튼을 한번씩 클릭 해 주고

오른쪽 상단의 탐색기 버튼을 누를 시 연결되어있는 폴더로 이동 됨

연결된 폴더에서 작업물을 알맞는 위치에 복사해 넣고

SourceTree에서 스테이지에 올라가지 않을 파일 항목에 복사해 넣은(혹은 수정한) 파일 목록이 잘 뜨는지 확인 후

"모두 스테이지에 올리기" 버튼 클릭

이후 아래의 Commit 메세지 작성 란에 간단한 설명을 작성하고 우측 하단에 Commit 버튼 클릭

마지막으로 좌측 상단의 Push 버튼 클릭
